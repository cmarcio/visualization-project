Introdução


Contexto

O Relatório Mundial da Felicidade é uma pesquisa publicada pela Rede de Soluções para o Desenvolvimento Sustentável da ONU e tem como objetivo medir a felicidade. O primeiro relatório foi publicado em 2012, seguido pelas peblicações de 2013, 2015, 2016 e 2017. O relatório de 2017 classifica 155 países de acordo com seu nível de felicidade e foi publicado dia 20 de março. O relatório continua crescendo e ganhando reconhecimento internacional, ajudando as autoridades e pesquisadores a entender como os indicadores sociais contribuem para a felicidade de uma nação.

Dados

O índice e ranking de felicidade usam dados fornecidos pelo Gallup World Poll. As pontuações são baseadas nas respostas das perguntas sobre avaliação da vida. Nesse tipo de questão os entrevistados são solicitados para que pensem em uma nota de 0 a 10 para sua vida, onde 0 é a pior tipo de vida que a pessoa possa imaginar e 10 a melhor tipo vida que ele consegue imaginar. A pontuação é gerada a partir de amostrar representativas de cada nacionalidade. Os outros seis atributos são estimativas do quanto cada um dos seis indicadores (Produção econômica, suporte social, expectativa de vida, liberdade, ausência de corrupção, generosidade) contribuem para tornar as avaliações da vida maior em cada país.

Atributo Descrição
País Nome do país
Posição Posição no Ranking de felicidade
Pontuação Índice de felicidade
PIB Medida do quanto o Produto Interno Bruto do país contribui para o índice de felicidade
Suporte social Medida do quanto o Suporte social contribue para o índice de felicidade
Expectativa de vida Medida do quanto a expectativa de vida contribui para o índice de felicidade
Liberdade Medida do quanto a liberdade contribui para o índice de felicidade
Generosidade Medida do quanto a generosidade contribui para o índice de felicidade
Ausência de corrupção Medida do quanto a ausência de corrupção contribui para o índice de felicidade
Resíduo de Distopia Medida do quanto o resíduo de distopia contribui para o índice de felicidade

O resíduo de distopia é o valor do índice de felicidade calculado para um país fictício onde todos os indicadores possuem o pior valor possível. A soma dos 6 atributos citados anteriormente mais o resíduo de distopia é igual ao índice de felicidade.

Pré-processamento

Para facilitar a utilização dos dados em uma aplicação Web os mesmos foram convertidos para o formato JSON (Javascript Object Notation) e estão disponíveis na pasta "public/data" no repositório do projeto. Alguns scripts de agregação de dados foram utilizados para associar a informação de continente aos países, para evitar perda de desempenho o resultado foi salvo em um novo arquivo que não precisa ser pré-processado pelos clientes quando fizer um acesso. Os scripts estão no arquivo "public/js/scripts.js" e os dados utilizados para agregar a informação de continentes aos países estão no site country.io.
